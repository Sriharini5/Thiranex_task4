# Task 4: Data Cleaning & Reporting Automation

## Project Overview

This project focuses on automating the data cleaning process and generating analytical reports using Python. The Online Store Orders dataset was cleaned by handling missing values, removing duplicate records, and standardizing data formats. Automated reporting and visualizations were created to gain meaningful insights from the dataset.

## Objectives

* Identify and handle missing values
* Remove duplicate records
* Standardize and clean data
* Generate automated data profiling reports
* Create visual summaries and insights
* Export the cleaned dataset

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* YData Profiling
* Google Colab

## Dataset

**Online Store Orders Dataset**

The dataset contains order-related information from an online store, including customer, product, sales, and transaction details.

## Data Cleaning Process

### 1. Data Loading

* Imported the dataset into Google Colab.
* Examined the structure and contents of the data.

### 2. Missing Value Handling

* Identified missing values.
* Filled numerical missing values using mean values.
* Filled categorical missing values with appropriate placeholders.

### 3. Duplicate Removal

* Detected duplicate records.
* Removed duplicate entries to improve data quality.

### 4. Data Standardization

* Cleaned column names.
* Standardized data formats and values.

## Automated Reporting

Generated an automated profiling report using YData Profiling to analyze:

* Data types
* Missing values
* Statistical summaries
* Correlations
* Dataset quality metrics

## Visualizations

* Histogram Analysis
* Correlation Heatmap
* Data Distribution Charts

## Project Outcomes

* Improved dataset quality and consistency.
* Generated an automated analytical report.
* Created meaningful visual insights for decision-making.
* Produced a cleaned dataset ready for further analysis.
